Enterprise AI Model

Associated with Cancer Institute NSW

Built a secure, privacy-first Enterprise AI system for Cancer Institute NSW that uses a fine-tuned Small Language Model, a cancer-specific Knowledge Graph, and a FAISS-backed retrieval layer to deliver accurate, evidence-linked answers grounded in internal PDFs, policies, and research material.

What I Built

I designed a hybrid knowledge engine where an ontology-driven Knowledge Graph captures entities like cancer types, treatments, risk factors, and outcomes, while a vector store retrieves the most relevant passages from documents. The model uses Retrieval-Augmented Generation to combine these signals and generate explanations with clear citations, keeping all data on-prem for compliance.

End-to-end, I implemented the pipeline from document ingestion and pre-processing through SLM fine-tuning, retrieval orchestration, and a simple UI that allows researchers and clinicians to ask natural-language questions and receive traceable, domain-aware responses.

Jio SecureID (Authentication)

Associated with Reliance Jio

Jio SecureID is a multi-tenant digital identity and MFA platform that delivers secure, low-latency authentication for enterprise clients at telecom scale. It supports high concurrency, strict access boundaries, and detailed auditability.

What I Delivered

I built and shipped core modules across the full stack, including Java Spring Boot microservices, REST APIs, and Angular-based portals for client management. I implemented secure client onboarding using mutual TLS and API key exchange, and worked with API Gateway controls such as allowlists to ensure only approved systems could access authentication endpoints.

Beyond the transactional path, I engineered telemetry and ETL pipelines that captured authentication and activity events for security analytics and audit logs, orchestrated using tools like Airflow and monitored through MLflow-backed anomaly detection. Services were containerised with Docker, deployed on Kubernetes, provisioned with Terraform and CloudFormation, and observed in production using Prometheus and Datadog to track latency, errors, throughput, and client activity.

Associated with Macquarie University
